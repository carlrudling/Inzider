Inzider - En Saas platform där content creators kan skapa och sälja reseguider och goto kollektioner. 

Den använder sig av följande teknologier:

Frontend:
● Next.js & TailwindCSS 


Backend: ● TypeScript 


Databas: ● Mongodb 

AI: ● Cursor 

CDN och bildlagring: ● Cloudflare 

Betalningshantering: ● Stripe Connect - Skapare kan ta betalt av köpare, platformen tar en procentuell avgift per transaktion 


Hosting: ● Digital Ocean droplets - Ej fullt skapat ännu
